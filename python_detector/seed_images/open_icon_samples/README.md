# Open Icon Training Samples

This folder contains small icon training samples collected from open-source icon sets that allow commercial use.

## Sources

- Heroicons: https://github.com/tailwindlabs/heroicons
  - License: MIT
  - Local license copy: `HEROICONS_LICENSE`
- Tabler Icons: https://github.com/tabler/tabler-icons
  - License: MIT
  - Local license copy: `TABLER_ICONS_LICENSE`

## Files

- `heroicons_solid/`: downloaded Heroicons solid SVG samples.
- `heroicons_outline/`: downloaded Heroicons outline SVG samples.
- `tabler_outline/`: downloaded Tabler outline SVG samples.
- `png/`: PNG thumbnails rendered from the SVG samples.
- `sheets/`: composite PNG sheets generated from the rendered PNG samples with varied colors, sizes, and backgrounds.

The generated PNG files are derived from the MIT-licensed SVG sources above.
